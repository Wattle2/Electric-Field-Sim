Electric Field Simulator

About

This is a web app that aids in visualizing a system of electric point charges and their electrostatic forces.
This app is built on a scalable, object-oriented backend. Intended for versatile use as a tool to help learners conceptualize electrostatics,
as well as a calculator capable of handling large datasets.

Navigation

The website title functions as a home button. Click the green buttons to move between parts of the simulation.

Inputting Data

When starting the simulation, the first input box will prompt for the number of charges present in the system.
You will then be prompted to enter the charge value (in Coulombs) and their X and Y values with respect to
an arbitrary origin. 1 graph unit is 1 meter.

Graph

The graph automatically normalizes all points to fit in the graph, but maintains relative distances and directions.
The arrows represent the direction of the net force on each respective point charge.

Data Output

Below the graph is a table of numerical data for each point charge, in the order they were inputted in.
The direction measurement follows the unit circle, where 180 degrees points left, and 0 degrees points right.
All data is rounded to 5 decimal places.

Calculation Method

The simulation uses Coulomb's law to find the force between two charges. This force then gets split into components and
added/subtracted with the components from subsequent Coulomb's law calculations. For example, in a system of 3 charges,
the components from charge 2 on charge 1 and charge 3 on charge 1 will be added together.

The Coulomb's law equation is given by:

$$ F = \frac{k \cdot |q_1 \cdot q_2|}{r^2} $$

Where:
k = 8.99 × 109 N m2/C2
q = Charge (in Coulombs)
r = Distance (in meters)
